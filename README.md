# Event Trigger Platform

A backend application for managing and triggering events based on user-defined conditions. Built with FastAPI, Docker, and PostgreSQL.

## Features

- Schedule and manage triggers (scheduled or API-based).
- Log triggered events with retention policies.
- Test API and scheduled triggers.
- Basic UI using Swagger/OpenAPI.

## Prerequisites

- Docker and Docker Compose installed.
